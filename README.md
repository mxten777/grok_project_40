## 📋 프로젝트 개요

강둑 송년회 웹사이트는 2025년 강둑 동기들의 특별한 송년회를 위한 종합 디지털 플랫폼입니다. 현대적인 웹 기술을 활용하여 참석자들이 편리하게 정보를 확인하고 소통할 수 있는 환경을 제공합니다.

### 🎯 주요 목표
- **디지털 초대장**: QR코드와 함께하는 현대적인 초대장 시스템
- **실시간 출석 관리**: 참석 현황을 실시간으로 확인하고 관리
- **추억 공유**: 갤러리를 통한 사진 공유 및 추억 저장
- **커뮤니티 형성**: 방명록을 통한 따뜻한 메시지 교환
- **모바일 최적화**: 모든 디바이스에서 완벽한 사용자 경험 제공

## ✨ 주요 기능

### 🏠 메인 페이지
- **화려한 히어로 섹션**: 그라데이션 배경과 애니메이션 효과
- **카운트다운 타이머**: D-DAY까지 남은 시간을 실시간 표시
- **이벤트 정보 카드**: 일시, 장소, 참석비 정보를 직관적으로 표시
- **빠른 액션 버튼**: 초대장 보기, 출석 체크 바로가기
- **기능 소개 섹션**: 각 페이지의 주요 기능을 미리보기

### 🎫 초대장 페이지
- **디지털 초대장**: 아름다운 디자인의 전자 초대장
- **QR코드 생성**: 초대장을 스마트폰으로 쉽게 공유
- **이벤트 세부정보**: 일시, 장소, 참석비 상세 안내
- **공유 기능**: 카카오톡, SMS 등 다양한 공유 옵션
- **반응형 디자인**: 모바일에서 완벽한 가독성

### 👥 출석 체크 페이지
- **실시간 출석 현황**: 참석/불참/미정 상태 표시
- **참석률 통계**: 시각화된 참석률 차트
- **참석자 목록**: 이름과 상태를 한눈에 확인
- **출석 상태 변경**: 본인의 출석 상태를 직접 업데이트
- **관리자 기능**: 출석 현황 관리 및 통계 분석

### 📸 갤러리 페이지
- **연도별 탭**: 2024년, 2025년 추억을 분류하여 표시
- **이미지 그리드**: 반응형 이미지 갤러리 레이아웃
- **이미지 확대**: 클릭 시 큰 이미지로 확대 표시
- **사진 업로드**: 새로운 추억 사진 추가 기능
- **무한 스크롤**: 많은 이미지도 부드럽게 탐색

### 💬 방명록 페이지
- **메시지 작성**: 따뜻한 응원 메시지 남기기
- **실시간 업데이트**: 새로운 메시지를 즉시 확인
- **프로필 이미지**: 각 메시지에 프로필 사진 표시
- **이모지 지원**: 다양한 이모지로 감정 표현
- **페이지네이션**: 많은 메시지도 효율적으로 관리

### 📅 행사 정보 페이지
- **상세 일정 안내**: 행사 세부 스케줄 및 타임테이블
- **장소 정보**: 지도 연동 및 교통편 안내
- **준비물 안내**: 참석 시 필요한 준비사항
- **연락처 정보**: 문의사항을 위한 연락처 제공
- **QR코드 안내**: 현장 등록을 위한 QR코드

## 🚀 설치 및 실행

### 사전 요구사항
- **Node.js**: 18.0 이상
- **npm**: 8.0 이상 (또는 yarn/pnpm)
- **Git**: 프로젝트 클론을 위해 필요

### 1. 프로젝트 클론
```bash
git clone <repository-url>
cd grok_project_30
```

### 2. 의존성 설치
```bash
npm install
# 또는
yarn install
# 또는
pnpm install
```

### 3. 환경 변수 설정 (선택사항)
프로젝트 루트에 `.env.local` 파일을 생성하고 필요한 환경 변수를 설정하세요.

```env
# 예시 환경 변수
NEXT_PUBLIC_KAKAO_MAP_API_KEY=your_kakao_map_api_key
NEXT_PUBLIC_SITE_URL=http://localhost:3000
```

### 4. 개발 서버 실행
```bash
npm run dev
# 또는
yarn dev
# 또는
pnpm dev
```

서버가 시작되면 [http://localhost:3000](http://localhost:3000)에서 웹사이트를 확인할 수 있습니다.

### 5. 프로덕션 빌드
```bash
npm run build
npm run start
```
